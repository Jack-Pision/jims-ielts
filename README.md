# JIMS IELTS

A premium IELTS training institute website built with Next.js 15, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4 + CSS Custom Properties
- **Animations:** Framer Motion, CSS @keyframes
- **State:** Zustand
- **Forms:** React Hook Form
- **Icons:** Lucide React

## Typography

- **Display:** Instrument Serif (Google Fonts)
- **Body/UI:** Inter (Google Fonts)

## Pages

- `/` - Home (Hero, Stats, Why JIMS, Destinations, Testimonials, Courses)
- `/courses` - Programs & Courses with filtering
- `/about` - About the founder, journey, student success, contact

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
